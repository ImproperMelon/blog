# CSS 的选择器事实上是条件语句
当你尝试过做一个简单的静态页面之后，你可能觉得 CSS 仅仅是：选择器 + 样式<br />但是在我们使用 scss 的时候，那种嵌套似乎把我们固定在了类选择器中，

事实上你明白：只要 CSS 选择器用得好，你可以获取到任何一个 HTML 元素，<br />也就是说： CSS 的选择器事实上是非常强大的。
